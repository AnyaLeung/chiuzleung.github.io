Fluent Python Overall

Posted on 2017-09-06

这本书主要着重Python作为编程语言的特性，以及一切Python库的介绍。对于具有Python2基础想学习Python3.x的读者是很友好的。内容不是特别适合初学者，更糟糕的是，可能导致初学者特殊方法和metaprogramming的滥用。（作者说的，雾

书主要包括六个部分。

第一部分

主要讲解了Python的data model以及特殊方法。

特殊方法（双下方法）用以让对象实现和支持一些语言框架，并与之交互：

  • 迭代
  • 集合类
  • 属性访问
  • 运算符重载
  • 函数和方法的调用
  • 对象的创建和销毁
  • 字符串表示形式和格式化
  • 管理上下文

特殊方法的存在是为了被Python解释器调用的，我们并不需要调用它们。


第二部分

包含了各种集合类型：sequence, set, mapping, 以及str 和bytes 的区别。

序列组成的数组

序列数据，包括字符串、列表、字节序列、数组等，都共用一系列操作：迭代、切片、拼接、还有排序。

可以将Python序列按其可变性分为可变序列和不可变序列，或分为扁平序列和容器序列。扁平序列只能保存数字、字符、字节等原子数据，但相对较快、简单。容器序列更为灵活，但出错几率更大。

列表推导和生成器表达式可以用以构建和初始化序列。

字典和集合

Python中在collections模块中提供现成的特殊映射类型，如defaultdict OrderedDict, ChainDict等。

大多数映射字典都提供了setdefault方法和update方法。前者用以更新字典里存放的可变值，后者让批量更新成为可能。

对于查找不到的key:

  • 创建defaultdict对象，给它配置一个味找不到的key提供默认value的方法
  • 特殊方法 missing ，通过这个方法自定义会发生什么。

    最后介绍了散列表的工作原理，和散列表带来的潜在影响。

文本和字节序列

从Python3的str对象获取的元素是unicode字符。

第三部分

把函数视为对象。
介绍了如何把函数作为一等对象使用，闭包，装饰器，以及可调用、函数属性、内省、参数注解和Python3里炘出现的nonlocal声明。

  • 函数的“一等”本性，即可以把函数赋值给变量，然后通过变量名调用。还可以把它作为参数传给其他函数，如map函数等。有了一等函数，就可以使用函数式风格编程，函数式编程的特点之一就是使用高阶函数。
    高阶函数是接受函数为参数，或者把函数作为结果返回的函数。如map, filter, sort, reduce函数等,Python3中，列表推导或生成器表达式具有map和filter的功能，且更容易阅读。
  • 介绍了Python3中的高阶函数。
  • 为了使用高阶函数，有时创建匿名函数更为便利。lambda关键字在Python表达式内创建匿名函数。
    除了用户的自定义函数，运算调用符也可以用在其他对象上，可以通过callable()内置函数判断对象是否可以调用。
  • 介绍了operator模块中的一些函数，以及functools.partial函数。
  • 随后介绍了Python中一些动态特征，尤其是一等函数。从“策略”模式开始，使用一等函数简化了实现方式。很多情况下，在Python中使用函数或可调用对象实现回调更为自然。
  • 最后主要讲解了闭包和装饰器。

第四部分

主要介绍类和面向对象。解释了引用的原理、“可变性”的概念，如何构建自定义集合类型等。

  • 每个Python对象都有标识、类型和值，只有对象的值会不时变化。
  • 如果两个变量指向的不可变对象具有相同的值，那么它们指代的对象是否是一个没有关系。（除不可变集合，如tuple和frozenset）。
  • 变量保存的事引用。
  • 如何用特殊方法和约定结构，定义行为良好且符合Python风格的类。
  • 讲解了vector，说明了slice的工作原理。

第五部分

集中在构造和库上，涉及到了generator、上下文管理器、协程，涵盖了yield from语法。以并发性和面向事件的I/O结尾。

第六部分

讲解了如何动态创建带属性的类，用以处理如JSON这类半结构化的数据。最后介绍类装饰器和元类这些概念。
